In 2016, Insider Tech (a U.S.-based technology news channel) published the following video, titled *"How to clone a security badge in seconds"*.

- The video demonstrates how to clone a security badge using a portable device. 
- It explains that RFID cards are commonly used for workplace access and can be easily copied by capturing the card's unique ID. 
- The video then shows how to modify a garage door reader to create a portable card cloner.
