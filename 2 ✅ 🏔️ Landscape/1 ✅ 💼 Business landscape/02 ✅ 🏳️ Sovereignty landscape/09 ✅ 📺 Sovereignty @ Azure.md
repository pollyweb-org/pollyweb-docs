In 2022, Microsoft Cloud published the following video, titled *"Satya Nadella keynote on Microsoft Cloud for Sovereignty and Azure confidential computing"*.

- This video is about Microsoft's new cloud offering, Microsoft Cloud for Sovereignty. 
- This offering is designed to help public sector customers meet their data sovereignty, security, and privacy requirements. 
- It is built on top of Azure Confidential Computing, which allows customers to encrypt their data while it is in use.
