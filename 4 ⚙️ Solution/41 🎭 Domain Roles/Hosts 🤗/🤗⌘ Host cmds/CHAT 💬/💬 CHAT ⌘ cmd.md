# 😃💬 Talker `CHAT` Command

> About
* Implemented by the [`.CHAT` 📃 script](<💬 CHAT 📃 script.md>)

## FAQ

1. **What's the CHAT command for?**

    The `CHAT` command
    * sets the [`$.Chat` 🧠 holder](<../../../../37 Scripts 📃/📃 Holders 🧠/System holders 🔩/$.Chat 💬/💬 $.Chat 🧠 holder.md>) holder
    * with a new or existing [Chat 💬](<../../../../35 💬 Chats/Chats 💬/💬 Chat.md>)
    * to provide language and format context for [Prompts 🤔](<../../../../35 💬 Chats/Chats 💬/🤔 Prompt.md>).

    ---
    <br/>

1. **What's the syntax?**

    ```yaml
    CHAT:
        Broker: <broker-name>
        Chat: <chat-uuid>
    ```

    Input|Purpose
    |-|-
    |`Broker`| [Broker 🤵 domain](<../../../../20 🧑‍🦰 UI/Brokers 🤵/🤵/🤵 Broker 🤲 helper.md>) name
    |`Chat`| [Chat 💬](<../../../../35 💬 Chats/Chats 💬/💬 Chat.md>) ID
