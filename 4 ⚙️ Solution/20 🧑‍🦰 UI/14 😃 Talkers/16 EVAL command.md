1. **What's an EVAL command?**

    An `EVAL` is a [Command](<10 Command.md>) that evaluates a [{Function}](<15 Function command.md>) into a placeholder.

    ---
    <br/>

2. **What's the syntax?**

    ```yaml
    - EVAL|{function} >> <placeholder>
    ```

    | Argument| Purpose
    |-|-
    | `{function}`| The [{Function}](<15 Function command.md>) to be evaluated.
    | `<placeholder>`| The placeholder to store the evaluation result.
