1. **What is the .Append function?**

    `{.Append}`

1. **What's the syntax of .Append?**

    ```yaml
    .Append(list, appendix)
    ```

    Input | Purpose | Example
    |-|-|-
    |`list` | Original string, | `AB` `$str`
    |       | or list of items |  `[A,B]` `$list`
    |`appendix` | string to append, | `CD`
    |           | or item to append, |  `C`
    |           | or items to merge | `[C,D]`

    |           | Merge  | `[1,2]` | `[3,4]` | `[1,2,3,4]`
